# 概要
これはAiWolfPyの動作確認環境のDockerイメージです。

# インストール

```
git clone https://github.com/MuNeNICK/docker-aiwolf.git
```

```
cd docker-aiwolf
```

```
docker-compose build
```

```
docker-compose up -d
```

# アクセス
ブラウザ上で
http://サーバIP:8080
でuBuntuのリモートデスクトップにアクセスできる。
